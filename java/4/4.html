# 连接池

1. jdbc连接池
2. 其实就是一个容器, 存放数据库连接的容器
3. 当系统初始化之后 容器被创建  容器中会申请一些连接对象
4. 当用户来访问数据库时 
5. 从容器中获取连接对象
6. 用户访问完之后, 会将连接对象还给连接池

:::tips 节约资源 用户访问高效 :::
